## 🛠️ 技术架构

### 前端技术栈
```
Vue 3.4.0          # 渐进式JavaScript框架
TypeScript 5.3.3   # 类型安全编程
Ant Design Vue 4.1 # UI组件库
Vue Router 4.2      # 路由管理
Pinia 2.1          # 状态管理
ECharts 5.4        # 数据可视化
D3.js 7.8          # 图形绘制
Vite 5.0           # 构建工具
Less 4.2           # CSS预处理器
```

### 项目结构
```
src/
├── layouts/           # 布局组件
│   └── MainLayout.vue # 主布局（侧边栏+顶栏）
├── views/            # 页面组件
│   ├── Dashboard.vue # 数据概览仪表板
│   ├── auth/         # 登录认证
│   ├── insights/     # 战略洞察模块
│   ├── opportunities/ # 机会引擎模块
│   ├── resources/    # 资源库模块
│   └── admin/        # 后台管理
├── router/           # 路由配置
├── styles/           # 样式文件
│   ├── global.less   # 全局样式
│   └── themes/       # 主题样式
└── main.ts          # 应用入口
```
